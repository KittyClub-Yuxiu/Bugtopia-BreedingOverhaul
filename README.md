# 繁育系统重构 (Breeding Overhaul)

**简介：**

你是否厌倦了原版概率低下的繁育概率？繁殖了十多只紫色的虫虫却一直不出金？

本 Mod 彻底重写了游戏的遗传逻辑，并可通过配置文件调整概率。

**核心功能：**

✅ **同稀有度**：
*   当虫虫稀有度相同时（比如两只白色，两只绿色），有 **50%** 的概率（可在配置文件中调整）生出下一稀有度的虫虫，否则保持当前等级。
*   原版中两只金色虫虫繁育会退化，现在会固定生出金色虫虫。

✅ **不同稀有度**：
*   当虫虫稀有度不同时（如白色和紫色，绿色和金色），你有 **20%** 的概率（可在配置文件中调整）生出下一稀有度的虫虫。
*   剩余概率会平均分配给两个虫虫稀有度之间的所有种类（如白色和紫色有20%概率获得金色，80%概率均分给除金色外的所有颜色）。

**【安装指南】**

1.  **安装 BepInEx**： 如果你是第一次使用 Mod，请先从 [BepInEx的GitHub发布页](https://github.com/BepInEx/BepInEx/releases) 下载最新版本的 BepInEx。
2.  **解压**： 将压缩包内的所有文件，解压到《虫虫生态箱》的游戏根目录 (也就是与 `Bugtopia.exe` 文件放在一起)。
3.  **安装本 Mod**： [下载 BreedingOverhaul.dll](https://github.com/KittyClub-Yuxiu/Bugtopia-BreedingOverhaul/releases)。
4.  **放入插件文件夹**： 将 `BreedingOverhaul.dll` 文件，放入游戏根目录下的 `BepInEx/plugins` 文件夹内。(如果 plugins 文件夹不存在，请先运行一次游戏，它会自动生成)。
5.  **完成！** 启动游戏即可享受！

**【如何修改配置】**

1.  先运行一次游戏，让 Mod 生成配置文件。
2.  进入 `BepInEx/config` 文件夹。
3.  找到名为 `BreedingLogicOverhaul.cfg` 的文件，用记事本打开。
4.  修改数值（`1.0` 代表 100%，`0.5` 代表 50%），保存即可。

**【注意事项】**

*   该 Mod 所有代码由 Gemini 3.0 Pro 生成，并经由本人多次测试。
*   直接删除dll文件即可恢复原版逻辑。
*   本 Mod 基于虫虫生态箱 1.3.6 开发，未来游戏更新可能会导致其失效，届时我可能会尝试更新。
*   建议使用 Mod 前建议备份一份存档文件以保证安全，创作者不对因安装 Mod 导致的存档丢失等各种可能性负责。
